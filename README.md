# BlockCredAI
Decentralized Employment Verification System with AI-Based Resume Fraud Detection. Blockchain (Solidity + Ethereum) + Next.js + FastAPI. College blockchain mini-project with smart contracts, NFT badges, and fraud detection ML model.

## How to run on localhost

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SaurabhForge/BlockCredAI.git
   cd BlockCredAI
   ```

2. **Install Dependencies in each folder:**
   ```bash
   # Install backend dependencies
   cd backend && npm install && cd ..
   
   # Install frontend dependencies
   cd frontend && npm install && cd ..
   ```

3. **Start the Frontend & Backend Servers:**
   You will need to open two separate terminals.
   
   **Terminal 1 (Backend):**
   ```bash
   cd backend
   npm run dev
   ```

   **Terminal 2 (Frontend):**
   ```bash
   cd frontend
   npm run dev
   ```

4. **View the Application:**
   Open [http://localhost:3000](http://localhost:3000) in your browser to see the frontend.  
   The backend API runs on `http://localhost:4000`.
